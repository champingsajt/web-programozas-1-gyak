A feladat rövid leírása: 

Olyan, megfelelő menüvel ellátott, több oldalból álló weblap készítése PHP, CSS, JavaScript (vagy más kliensoldali függvénykönyvtár, pl. jQuery) segítségével, amely az oldal működése során AJAX/fetch-megoldást is alkalmaz. Az oldalnak legalább alapszintű felhasználókezelést is meg kell valósítania (ki/bejelentkezés/regisztráció). Az alapvető felhasználókezelés magában foglalja a jelszavak biztonságos tárolását (pl. hash-eléssel) és a munkamenet-kezelést.

A megoldáshoz keretrendszer, tartalomkezelő rendszer nem használható. A szerveroldalon a PHP és a MySQL használata elvárt.

A fejlesztés során törekedni kell a tiszta, olvasható kódra, a megfelelő kommentezésre és a biztonsági szempontok (pl. SQL injection, XSS elleni védelem) figyelembevételére.



A honalp témája : Procrastinator exciter 

A "Procrastinator exciter" téma keretében a weboldal célja, hogy segítse a felhasználókat a halogatás leküzdésében. Ez magában foglalhatja például feladatok, célok vagy motivációs tartalmak kezelését. A MySQL adatbázis tervezésekor vegyék figyelembe a felhasználók, feladatok és egyéb releváns adatok tárolásának szükségességét.

Példák a `example` mappában: A mappa mintákat vagy inspirációt tartalmazhat a feladat megvalósításához, például UI/UX ötleteket vagy technikai megközelítéseket.
