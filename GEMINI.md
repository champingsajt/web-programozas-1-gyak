Web Programming I.



Task to be submitted – Spring 2025/26



Short description of the task:



Creating a multi-page website with an appropriate menu using PHP, CSS, JavaScript (or other client-side function library, e.g. jQuery), which also uses an AJAX/fetch solution during the operation of the page. The page must also implement at least basic user management (logout/login/registration). Basic user management includes secure storage of passwords (e.g. by hashing) and session management.



A framework or content management system cannot be used for the solution. PHP and MySQL are expected to be used on the server side.



During development, efforts should be made to ensure clean, readable code, proper commenting, and security considerations (e.g. protection against SQL injection, XSS).



The theme of the honalp : Procrastinator exciter



Within the framework of the "Procrastinator exciter" theme, the website aims to help users overcome procrastination. This could include, for example, managing tasks, goals, or motivational content. When designing the MySQL database, consider the need to store users, tasks, and other relevant data.



Examples in the `example` folder: The folder can contain samples or inspiration for implementing the task, such as UI/UX ideas or technical approaches.
